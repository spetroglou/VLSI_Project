# VLSI_Project

This is the semester project that I implemented in the "Digital Systems VLSI" course. The goal of this project was to design and characterise 3 standard cells: the XOR and XNOR logic gates, and a D-Flip Flop. To design these cells I used the "Magic VLSI" software tool and I verified their functionality utilizing the "NG-Spice" circuit simulator tool. When the verification was done, I measured the timing arcs, including cell rise/fall times and rise/fall transition times, from all standard cells, and the setup and hold constraint arcs from the D-FF circuit. The results were stored in json configuration files in the folder named "vlsi_measurements". Finally, I created the script named "char_flow.py" that is parsing all the data from the json files and creates a library file using the Synopsys Liberty Standard convention.
